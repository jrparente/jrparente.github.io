Creating Your Personal GitHub Page

Step 1: Repository Setup

  1. Sign in to your GitHub account or create one if you haven't already.
  2. Click on the 'New' button in the top right to create a new repository.
  3. Name your repository as <username>.github.io. Replace <username> with your GitHub username.
  4. Choose 'Public' so that your page is accessible to everyone.
  5. Click 'Create repository.'

Step 2: Create and upload Your Website Files

  1. In your computer, create an 'index.html' file for your website. A very basic website structure would be: 
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Your Name</title>
    </head>
    <body>
        <h1>Your Name</h1>
        <img src="your-image.jpg" alt="Your Name">
    </body>
    </html>
  1. Navigate to the repository you have created in the previous step, and click the top right button to 'Add file' and select 'Upload files'.
  2. Upload your 'index.html' file and the image you used to your repository. Make sure the image is in the same folder as the index.html file.
  3. Commit the changes.

Step 3: Check Your Page

  1. After a few moments, your page will be accessible at https://<username>.github.io.

Optional Enhancements

You can further customize your page by editing the HTML and CSS in your 'index.html' file.

For a very simple way of adding some styling to your page, you can add styles to the head of your index.html file. For example:

    <style>
        body {
            max-width: 1200px;
            margin: 2rem auto;
            padding: 0 1rem;
            
            display: flex;
            flex-direction: column;
            align-items: center;
            justify-content: center;

            font-family: Verdana, Geneva, Tahoma, sans-serif;
            line-height: 1.5;
        }
    
        img {
            display: block;
            width: 100%;
        }
    
        h1 {
            text-align: center;
        }
    </style>
